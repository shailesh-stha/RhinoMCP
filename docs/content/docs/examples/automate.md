One-shot chores. Cleanup, renaming, layer housekeeping &mdash; the kind
of thing you'd write a script for if you did it often enough.

## Try this prompt

{{< prompt >}}
Rename every layer to title case, delete the empty ones, and move any
off-layer objects to a layer called `Imported`.
{{< /prompt >}}

## What you should see

The assistant works through your layer table, reports what it changed,
and leaves the document in a tidier state. Undo is one step.
