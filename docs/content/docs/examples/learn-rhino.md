Use the assistant as a tutor that already knows what's in your document.
Better than searching the help when you can point at the actual geometry
and ask.

## Try this prompt

{{< prompt >}}
How do I make a fillet between these two surfaces? Walk me through it,
and do the first step for me so I can see what it looks like.
{{< /prompt >}}

## What you should see

A short explanation of the command, then the assistant runs it on the
selected surfaces so you have a concrete result to compare against. Ask
follow-ups to keep learning: "what's the difference between Fillet and
Blend?", "why did that one fail?".
